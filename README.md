## 🎯 项目概述

**卓优：中华版**是经典战略棋盘游戏"Acquire"的现代化网页实现，融入了独特的中式工业主题。玩家扮演企业大亨的角色，在 7 家不同主题的中式公司中建立工业帝国，通过战略性的板块放置、股票投资和企业并购来争夺市场主导地位。

### 游戏主题

以现代中国快速发展工业格局为背景，玩家建立和收购代表不同行业的企业：

- 🏭 **黄山纸业** (HuangShan Paper) - 黄色
- 🌌 **蓝天纺织** (BlueSky Textile) - 蓝色
- ☀️ **朝阳五金** (ChaoYang Hardware) - 橙色
- 🚩 **红旗造船** (RedFlag Shipbuilding) - 红色
- 💎 **紫金仪表** (ZiJin Instruments) - 紫色
- ⚛️ **核能重工** (Nuclear Heavy Industry) - 绿色
- 🌳 **橡树日化** (Oak Daily Chemicals) - 青色

## ✨ 核心功能

### 🎮 核心玩法

- **3-6 人多人游戏**: 支持 3 至 6 名玩家实时对战
- **战略性板块放置**: 在 9x12 网格上建立企业版图
- **动态股票市场**: 买卖股票，实时价格波动
- **企业并购**: 执行战略性收购和合并
- **安全企业保护**: 拥有 11 块以上板块的企业免疫并购

### 🖥️ 用户界面

- **现代化 React 界面**: 使用 React 和 Tailwind CSS 构建的简洁专业界面
- **实时更新**: 游戏状态在所有玩家间即时同步
- **交互式游戏棋盘**: 企业版图的可视化呈现
- **玩家仪表板**: 追踪现金、持股和表现
- **股票市场视图**: 实时股票价格和企业信息
- **回合日志**: 游戏操作的完整历史记录

### 🌐 多人游戏功能

- **基于房间的多人游戏**: 创建或加入私人游戏房间
- **实时通信**: 基于 Socket.io 的即时更新
- **主持人权限系统**: 主持人管理房间和游戏状态
- **连接状态**: 网络连接状态的可视化指示

### 📱 响应式设计

- **跨平台兼容**: 支持桌面、平板和移动设备
- **响应式布局**: 适配不同屏幕尺寸
- **触屏友好控制**: 针对移动游戏体验优化

## 🎲 游戏机制

### 游戏阶段

1. **板块放置**: 放置板块来建立或扩张公司
2. **并购处理**: 处理企业收购和分红
3. **股票购买**: 每回合最多购买 3 股股票
4. **手牌补充**: 从牌库抽取新板块

### 胜利条件

- 任何公司达到 41 块板块
- 所有公司都成为"安全企业"（11 块以上）

### 股票系统

- 每家公司共有 25 股可购买
- 根据公司规模和等级动态定价
- 大股东（10 倍股价）和小股东（5 倍股价）
- 合并期间自动股票转换

### 企业等级

- **一级**: 基础价格$200（黄山纸业、蓝天纺织）
- **二级**: 基础价格$300（朝阳五金、红旗造船、紫金仪表）
- **三级**: 基础价格$400（核能重工、橡树日化）

## 🚀 安装指南

### 环境要求

- **Node.js** 19.2 或更高版本
- **npm** 或 **yarn** 包管理器
- **现代网页浏览器** (Chrome, Firefox, Safari, Edge)

### 快速开始

1. **克隆代码仓库**

   ```bash
   git clone <repository-url>
   cd acquire_-china-edition
   ```

2. **安装依赖**

   ```bash
   npm install
   ```

3. **启动开发服务器**

   ```bash
   npm run dev
   ```

4. **在另一个终端中，启动多人游戏服务器**

   ```bash
   node server.js
   ```

5. **打开浏览器**
   访问 `http://localhost:3000`

### 生产构建

```bash
# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 🎮 使用说明

### 开始游戏

1. **进入大厅**

   - 输入你的玩家名称
   - 选择创建新房间或加入现有房间

2. **创建房间** (主持人)

   - 点击"创建新房间"
   - 与其他玩家分享房间 ID
   - 等待玩家加入

3. **加入房间** (玩家)

   - 输入主持人提供的房间 ID
   - 等待主持人开始游戏

4. **游戏流程**
   - 跟随屏幕上的回合指示器
   - 战略性放置板块来建立或合并公司
   - 通过购买股票管理你的投资组合
   - 目标是在游戏结束时拥有最多的现金

### 操作控制

- **点击** 在游戏棋盘上放置板块
- **选择** 手牌中的板块来放置
- **购买股票** 在股票购买阶段
- **查看规则** 使用帮助按钮获取详细说明

## 🌐 多人游戏设置

### 本地开发

默认设置使用 `localhost:3000` 作为前端和后端。只需运行开发服务器和 Socket.io 服务器即可。

### 生产部署

1. **前端部署**

   - 构建项目：`npm run build`
   - 将 `dist` 文件夹部署到你的 Web 服务器

2. **后端服务器**

   - 将 `server.js` 部署到 Node.js 托管服务
   - 更新 `utils/multiplayer.ts` 中的 Socket.io URL 指向你的生产服务器

3. **环境配置**
   ```typescript
   // 在 utils/multiplayer.ts 中更新
   const SOCKET_URL = "https://your-production-server.com";
   ```

### 网络要求

- WebSocket 支持实现实时多人游戏
- 每局游戏最少 3 名玩家，最多 6 名玩家
- 建议使用稳定的互联网连接
